## Tuner

* -> tuneIn    - have the tuner start up and "tune in"
* -> start     - alias for tuneIn
* -> stop      - have the tuner disconnect, shut down, etc.
* -> state     - notStarted, tuned, lost, error, stopped
* [ev] tuned   - tuner is tuned in (or tuned in again)
* [ev] message - message received by the tuner, with message content
* [ev] lost    - tuner lost its connection (or analogue thereof)
* [ev] error   - error

## Repeater

* -> addTuner
* -> start
* -> turnOn  - alias for start
* -> stop
* -> turnOff - alias for stop
* -> state   - stopped, running, degraded, failed
